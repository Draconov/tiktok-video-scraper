# Contributing

1. Fork the repository.
2. Create a focused branch.
3. Keep the script compatible with Windows PowerShell 5.1.
4. Do not commit binaries, cookies, downloaded media, or personal logs.
5. Run the GitHub Actions syntax check.
6. Explain behavior changes in the pull request.

TikTok and yt-dlp behavior changes frequently. Bug reports should include the yt-dlp version, selected channel, browser mode, and a sanitized log.
